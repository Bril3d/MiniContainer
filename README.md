# 🚀 MiniContainer

> Run dev environments instantly — no Docker complexity, no heavy setup.

MiniContainer is a **lightweight, fast container management tool** built on top of Podman. It provides a simple CLI and intuitive GUI to run containers and development environments without the resource overhead of Docker Desktop.

## ✨ Features

- 🏃 **One-command environments** — `mini run python`, `mini run postgres`
- ⚡ **No daemon** — zero idle CPU/RAM usage
- 📦 **Presets** — pre-configured environments for Python, Node, Postgres, MongoDB, Jupyter
- 🔍 **Smart detection** — auto-detects project type from your files
- 📊 **Resource monitoring** — real-time CPU/RAM usage per container
- 🖥️ **Desktop GUI** — beautiful Tauri-based interface (coming soon)

## 📋 Requirements

- [Podman](https://podman.io/getting-started/installation) installed
- Windows (with WSL2) or Linux
